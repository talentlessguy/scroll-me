А. Сложение двух чисел

Ограничение времени 1 секунда
Ограничение памяти 64Mb
Ввод стандартный ввод или input.txt
Вывод стандартный вывод или output.txt

Нужно сложить два числа

Формат ввода

В первой строке дано число a, во второй число b.

Формат вывода

Выведите сумму двух чисел

Пример 1

Ввод Вывод 
1
0
 
1

Пример 2

Ввод Вывод 
1
2

3
 
#include <iostream>

using namespace std;

int main () {
  int a, b;

  cin >> a;

  cin >> b;


}
